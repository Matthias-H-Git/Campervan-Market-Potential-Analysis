# Campervan Market Potential Analysis

## 📌 Project Overview

Ziel dieses Projekts ist die Analyse des aktuellen sowie die Prognose des zukünftigen Marktpotentials an Wohnmobilen in einem geographischen vom Kunden definierten Zielmarkt in Bayern auf Basis öffentlich verfügbarer statistischer Daten.

Dieses Projekt entstand im Rahmen eines freiberuflichen Beratungsauftrags zur Bewertung regionaler Marktpotenziale. Sie stellt einen Teil eines größeren Business Cases dar, die anderen Teile und relevanten Ableitungen unterliegen jedoch der Geheimhaltung gemäß einer NDA mit dem Kunden.

---

## 🎯 Key Questions

- Wie groß ist das Marktpotential für Wohnmobile im Zielmarkt 2025?
- Wie entwickelt sich das Marktpotential bis 2040?
- Welche Faktoren beeinflussen das Marktpotential?
  
---

## 📊 Methods

- Marktpotentialanalyse
- Marktpotentialprognose: Lineare Regression
- Plausibilisierung mittels externer Daten
- Regressionsanalyse 

---

## 🧠 Key Insights

- Zielregion besitzt erhebliches Wachstumspotenzial
- Bisherige Entwicklung entspricht den stärksten Wohnmobilregionen Deutschlands
- Wirtschaftskraft erklärt die Wohnmobildichte nicht
- Weitere Einflussgrößen bestimmen die regionale Nachfrage
